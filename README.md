# # 📊 Sales Analytics Pipeline using Google Cloud Platform

## 🚀 Project Overview

This project demonstrates an end-to-end data analytics pipeline built using Google Cloud Platform (GCP). The goal of the project is to analyze sales data and generate actionable business insights such as revenue trends, top-performing products, and high-value customers.

---

## 🎯 Business Problem

Organizations need to understand their sales performance to make data-driven decisions. This project answers key business questions:

* Which products generate the highest revenue?
* Who are the most valuable customers?
* What is the overall sales performance?

---

## 🛠️ Tech Stack

* Google Cloud Platform (GCP)

  * BigQuery (Data Warehouse)
  * Cloud Storage (Data Ingestion)
  * Looker Studio (Data Visualization)
* SQL (Data Analysis)

---

## 🏗️ Architecture

CSV File → Cloud Storage → BigQuery → SQL Analysis → Looker Studio Dashboard

---

## 📂 Dataset

* Sales dataset containing:

  * Customer ID
  * Product
  * Quantity
  * Price

---

## 📊 Key Analysis Performed

* Revenue Calculation using SQL
* Product-level performance analysis
* Customer segmentation based on spending
